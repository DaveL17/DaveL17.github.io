Indigo Home Control Server

  • Indigo - Indigo Home Control Server is the preeminent Mac-based Smart Home Hub. It works natively with Insteon, X10 and Z-Wave and is extensible to include many other protocols. In addition, it provides a facility to run Python scripts and Applescript and maintains a system that supports official and third-party plugins. The repositories referenced below are my plugins and scripts for Indigo. You can find more information at:

    Indigo Domotics

  • Indigo Forum - I have a moderated forum on the Indigo Domotics User Support Forums which is the primary place to post questions, comments, and feature requests.

Plugins for Indigo

  • BikeShare Plugin - The BikeShare plugin allows users to create Indigo Home Control Server devices representing Bike sharing service locations.

  • GhostXML Plugin - The GhostXML plugin allows XML and JSON data sources (either local or on the internet) to populate device states in an Indigo device.

  • Matplotlib Plugin - The Matplotlib plugin allows users to easily create attractive charts for display on Indigo Home Control Server control pages. The plugin also includes a facility to create necessary CSV data for display.

  • OWServer Plugin - The OWServer plugin allows users to connect the Embedded Data Systems (EDS) 1-Wire Server to the Indigo Home Control Server. The user creates devices that represent individual 1-Wire devices.

  • Thingspeak Plugin - The Thingspeak Plugin for the Indigo Home Control Server provides a facility to upload Indigo data elements to Thingspeak. It supports both remote and locally-hosted Thingspeak servers.

  • WUnderground Plugin - The WUnderground plugin allows users to create Indigo Home Control Server devices which provide a facility to create individual weather devices that will be linked to Weather Underground.

  • WUnderstation Plugin - The WUnderstation Plugin for the Indigo Home Control Server (Indigo) is used to provide a bridge between Indigo and the Weather Underground Personal Weather Station network data system.


Scripts for Indigo

  • Animated GIF for Control Page - This script generates animated GIFs like scrolling text for Indigo Control Pages.

  • Battery Charting Script - This script identifies all battery-powered devices and generates a bar chart which shows the battery life remaining.

  • Calendar Image Generator Script - This script creates a small image depicting the current month.

  • Flat File Length Management - A simple script to maintain CSV flat files for length (retain X lines).
  • Gephi and Indigo - Gephi is visualization and exploration software for all kinds of graphs and networks. Gephi is open-source and free. I use Gephi to map my Z-Wave network as a visual display of the network mesh.

  • Gnuplot Scripts - Gnuplot is a portable command-line driven graphing utility for OS X and many other platforms. The source code is copyrighted but freely distributed (i.e., you don't have to pay for it). It was originally created to allow scientists and students to visualize mathematical functions and data interactively, but has grown to support many non-interactive uses such as web scripting. It is also used as a plotting engine by third-party applications like Octave. I tend to use Matplotlib but maintain some Gnuplot scripts on the Indigo forums.

  • Low Battery Notification Script - As the name suggests, this simple script is used to create notifications when battery-powered Z-Wave device batteries become low.



  • Matplotlib Wind Direction Script - Use this script to create a directional needle graphic for displaying wind vectors, and is designed specifically to be used with the WUnderground Gauge Pack. It can be used independently as well. While not as attractive as the needle graphic included with the WUnderground Gauge pack (which uses 360 images for wind direction alone), it is fairly serviceable.

  • Target Humidity Script - Use this script to control humidity delivery in a home HVAC system. It provides more granular control over humidity levels rather than the 5 percent increments that are typical with residential humidistats. Often, simple humidistats control humidity in increments of 5 or even 10 percent--sometimes even "low, medium, or high". By connecting a relay module to the humidifier in place of the humidistat, you can use this script to maintain fine-level (and automated!) control of humidity as the temperature fluctuates.

  • WUnderground Gauge Pack - Not a script or plugin per se, but a collection of images that are designed to be used with the WUnderground plugin to create complex image-based gauges to display weather-related information. The images here reflect just one of the available themes - the Aviation theme - which is meant to mimic the instrumentation found on old-timey airplanes.

  • ZWave Node Matrix - This script creates an image depicting a Z-Wave node matrix. That is, a graphical representation of each Z-Wave device's negihbor list. It has features to display additional information such as which devices are battery-powered, which devices have been offline for a set amount of time, and so on.
